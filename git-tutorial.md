## GitHub Setup
1. Make new repository (click new on github)
2. Add Remote (git remote add origin (link))
## Git Setup
1. Git Init (`git init`)
2. Git Add (`git add .`)
3. Git Commit (`git commit -m "message"`)
4. (Optional Git Push) (`git push origin master`)
5. Go back to step 2
